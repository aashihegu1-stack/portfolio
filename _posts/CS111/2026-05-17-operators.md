## What are Operators?

Operators are symbols that perform operations on values. They are the verbs of programming — they do things.

## Arithmetic Operators

```python
a + b    # addition
a - b    # subtraction
a * b    # multiplication
a / b    # division (float)
a // b   # integer division
a % b    # modulo (remainder)
a ** b   # exponent
```

## Comparison Operators

These return `True` or `False`.

```python
a == b   # equal
a != b   # not equal
a < b    # less than
a > b    # greater than
a <= b   # less than or equal
a >= b   # greater than or equal
```

## Logical Operators

```python
True and False   # False
True or False    # True
not True         # False
```

## Assignment Operators

```python
x = 5
x += 2   # same as x = x + 2
x -= 1
x *= 3
```

## Why it matters

Operators are how you express almost every calculation, comparison, and decision in a program. They look small but show up in nearly every line of code.
